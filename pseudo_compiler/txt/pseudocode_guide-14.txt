Composite data type

A composite data type is a collection of data that can consist of different data types, grouped under one

identifier. The composite type should be declared as follows:

TYPE ← identifier1>
DECLARE ← identifier2>
DECLARE ← identifier3>

ENDTYPE ← data type>
